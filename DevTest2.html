Ayuda a crear una documentación más clara y efectiva.

Hola! En esta seccion encontraras como tu puedes contribuir a la documentación de la plataforma y a la creacion de componentes reutilizables.

Primero conoce a las dos etiquetas principales.

code-net-driven: utilizala cuando quieras mostrar sola una porcion de codigo.

tabs-net-driven utilizala cuando quieras mostrar tu HTML, SCSS y JavaScript en un tab de navegacion.

Como usar code-net-driven:

Primero crea un modulo html y pega el componente ya creado o pon una imagen, video o link del mismo.

Segundo envuelve el codigo en la etiqueta code-net-driven e indicar el lenguaje. Aqui te muestro un ejemplo:

window.addEventListener("load", function() { var menuItems = document.querySelectorAll("#menuItem13724"); menuItems.forEach(function(menuItem) { if (menuItem.tagName === 'A') { menuItem.target = "_blank"; } }); });

Tercero pega tu codigo en un HTML module con formato HTML. Si pegas tu codigo con formato RAW. El coigo HTML se puede mostrar en una sola linea.

Como usar tabs-net-driven

Primero crea un modulo html y pega el componente ya creado o pon una imagen, video o link del mismo.

Segundo envuelve el codigo en la etiqueta code-net-driven e indicar el lenguaje. Aqui te muestro un ejemplo:

Button1 Button2 Button3
.homepage-options { display: flex; flex-wrap: wrap; flex-direction: row; justify-content: center; a { padding: 1em; margin: 5px; flex: 1 1 auto; max-width: 250px; text-align: center; background-color: #007bff; color: white; text-decoration: none; font-weight: bold; transition: background-color 0.4s; &:hover { background-color: #0056b3; } } } // No se requiere JavaScript para este componente

Y listo! Tu componente ya esta listo para ser usado.
